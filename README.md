# Python Algorithms

### Motivation
* Practice Algorithms
* It's more fun when you can see them

### Usage

```
$ python -m algos --help
$ python -m algos sort bubble
$ python -m algos sort selection --speed 0.2 --height 10 --width 10
$ python -m algos brackets "((((((())[])))"
```
